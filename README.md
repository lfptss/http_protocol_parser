# HTTP Protocol Parser (Coded with iRule) --*unfinished*

## Introduction

HTTP Protocol Parser is a HTTP parser implemented using BIG-IP iRule, used to simulate the basic functions of an http profile. With this set of scripts, you can achieve similar functionality without needing to mount an http profile on **Virtual Server**. 

这套脚本是使用BIG-IP iRule实现的http协议解析器，用来模拟http profile的基本功能。在挂载这套脚本后，你可以在virtual server上不挂载http profile就能获得类似功能。

**IMPORTANT !!!** 

This set of scripts is only for learning iRules, HTTP, TCP, and related knowledge, and must not be used in a production environment. If there is a need for HTTP protocol parsing, please use the http profile.

这套脚本只能用来学习iRules，HTTP，TCP和相关的内容，不能用于生产环境，如果需要使用http协议解析的功能，请使用http profile。
